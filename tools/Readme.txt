概要
1つの画像データを画像処理により水増しするツール
 - 1枚の画像→88パターンの画像に
　　(オリジナル1+回転6+射影4) x (無変換1+ガンマ2+平滑化2+Salt&Pepper3)
　 注)
     - 入力画像はjpg, pngのみ.
     - 出力画像はjpg.

 - "-cnv"：プログラム中の画像変換パラメータに基づいて, 学習用サイズの画像に変換
   注)
     - オブジェクトサイズは画像変換が入るので, オブジェクトより
       一回り大きいサイズを指定する.

使い方
　python create_samples.py <-cnv> input_dir
